# Armário Inteligente de Medicamentos

Projeto desenvolvido para a disciplina de Sistemas Embarcados.

## Integrantes

- Isabela Silva
- Júlia Lourenço
- Livia Amaral
  
## Objetivo

Desenvolver um sistema inteligente capaz de lembrar o usuário do horário correto dos medicamentos utilizando Arduino, Bluetooth, LEDs, buzzer e botão de confirmação.

## Componentes

- Arduino Uno
- HC-05
- Potenciômetro
- LEDs
- Buzzer
- Botão
- Resistores

## Funcionalidades

- Comunicação Bluetooth
- Alerta sonoro
- Alerta visual
- Sistema de pontuação
- Filtro de média móvel
- Rotina de teste
